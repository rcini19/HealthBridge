Platform Guidelines for Review

As an administrator, familiarize yourself with these guidelines to ensure consistent and fair review of donations and requests.

Donation Guidelines

Medicine Quality Requirements

  • Medicine must be in its original, unopened packaging
  • No expired or near-expiry (within 30 days) medicines accepted
  • Clear, legible labels showing medicine name, dosage, and expiry date
  • No damaged, torn, or tampered packaging
  • Prescription medicines must include proper documentation if required

Photo Requirements

  • Upload clear, well-lit photos of the medicine
  • Show the complete packaging including expiry date
  • Include multiple angles if necessary
  • Ensure all text on packaging is readable in photos

Information Accuracy

  • Provide accurate medicine name and generic name
  • Enter correct dosage strength (e.g., 500mg, 10ml)
  • Specify exact quantity available
  • Verify expiry date is accurate and clearly visible
  • Include manufacturer name if possible

Prohibited Items

  • Controlled substances without proper authorization
  • Opened or partially used medicines
  • Medicines stored improperly (not at recommended temperature)
  • Counterfeit or unverified medicines
  • Medicines without clear labeling

Request Guidelines

Valid Request Requirements

  • Only request medicines genuinely needed for treatment
  • Provide accurate reason for medicine request
  • Request appropriate quantities based on actual need
  • Verify medicine name and dosage match prescription
  • Check medicine availability before making a request

Prescription & Documentation

  • Upload valid prescription if requesting prescription medicines
  • Ensure prescription is current and not expired
  • Prescription must clearly show medicine name and dosage
  • Provide any additional medical documents if required
  • Keep original documents for verification if requested

Request Information

  • Clearly state medical condition or reason for request
  • Specify exact quantity needed and duration of treatment
  • Provide contact information for delivery coordination
  • Be responsive to administrator questions or requests
  • Update request status promptly

Prohibited Actions

  • Requesting medicines for resale or unauthorized distribution
  • Making duplicate requests for the same medicine
  • Requesting excessive quantities beyond medical need
  • Providing false medical information
  • Claiming medicines without intent to pick up

Administrator Review Standards

  • Be thorough: Review all photos, documents, and information provided
  • Be fair: Apply guidelines consistently to all users
  • Be clear: Provide specific reasons when rejecting submissions
  • Be prompt: Review submissions in a timely manner
  • Be vigilant: Watch for suspicious patterns or fraudulent activity
  • Be helpful: Guide users to correct minor issues when possible
  • Document decisions: Keep clear records of approval/rejection reasons
